Claridad para conversaciones incómodas (100% local)

Instrucciones rápidas

Las parejas llegan a un momento en que es necesario enfrentarse a la verdad de las emociones y sentimientos. Y aunque duela, es mejor saber a qué enfrentarnos en lugar de disfrazar, esconder o suponer las verdades propias y de la otra persona. Intenta ser lo más honesta(o) respondiendo estas preguntas. Son sólo para ti, y para que organices ideas y sentimientos de cara a las conversaciones incómodas que hay que tener en la relación.

  • Esta app funciona sin servidor. Puede guardar tus respuestas en este dispositivo (opcional).
  • Si te activas mucho, pausa y vuelve cuando estés listo/a.
  • Esto no es para ganar una discusión: es para aclararte y llegar con propuestas concretas.
Al pulsar, baja automáticamente al cuestionario.
Consejo: si compartes el dispositivo con otra persona, usa un perfil distinto del navegador o desactiva el guardado.

Historial

—

M1. Radar rápido

Para qué sirve: ubicarte hoy (sin justificarte). Te ayudará a ver cambios con el tiempo.

  • Mueve las barras de 0 a 10.
  • Responde pensando en hoy, no en “cómo debería ser”.

Escalas 0–10. 0 = nada / 10 = muchísimo.

M2. El ciclo que nos mantiene mal

Para qué sirve: describir el patrón que mantiene el problema.

  • No es para culpar, es para entender cómo se alimenta el conflicto.
  • Busca frases concretas, sin “siempre / nunca”.

Describe el patrón. La idea es ver cómo se mantiene el problema, no “quién es el culpable”.

M3. Lo que sí funcionó

Para qué sirve: rescatar lo valioso sin negar lo difícil.

  • Esto baja la hostilidad y ayuda a hablar con más claridad.

Esto no borra lo difícil. Solo evita que la conversación sea una demolición.

M4. Temas ineludibles

Para qué sirve: aterrizar los temas que no pueden seguir debajo de la alfombra.

  • Cada tema tiene preguntas específicas.
  • Responde para ti, no para convencer al otro.

Por cada tema: gravedad, necesidad, oferta, propuesta y límite.

M5. Ruta (decisión o plan de transición)

Para qué sirve: elegir una ruta o dejar condiciones claras para decidir.

  • Si eliges “pausa”, define reglas y fechas de evaluación.

Puedes elegir una ruta hoy o dejar condiciones para decidir.

M6. Guion de conversación

Para qué sirve: tener un orden para hablar sin quedar atrapados en reproches.

  • Si la conversación se calienta, usa una pausa y define dónde se queda para retomar sin empezar de cero.

Un orden sugerido para no caer en reproches infinitos.

  1. 10 min: reconocimiento breve de lo que sí funcionó (1–2 frases cada uno).
  2. 20 min: cada uno expone sus 3 temas ineludibles (sin interrumpir, sin debatir).
  3. 20 min: cada uno hace propuestas (no quejas). “Necesito / ofrezco / propongo / mi límite”.
  4. 10 min: eligen una ruta (separación / pausa / reintento) o el siguiente paso con fecha.
  5. Cierre: un acuerdo mínimo para proteger al niño (no usarlo de mensajero, no discutir delante de él).

Privacidad (100% local)

Esta app no envía tus respuestas a ningún servidor. Si activas el guardado, se guardan en tu navegador (este dispositivo).

  • Si otra persona usa este mismo dispositivo/perfil del navegador, podría verlas.
  • Puedes desactivar el guardado o borrar todo cuando quieras.
  • “Exportar informe” genera un documento en tu equipo. Compártelo solo si tú quieres.
