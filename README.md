# Brújula – Conversación decisiva (100% local)

App simple (sin backend) para que cada persona llegue a una conversación decisiva con claridad:
- Radar emocional
- Ciclo que mantiene el problema (enfoque estratégico)
- Gratitud / aprendizajes
- Temas ineludibles (necesito / ofrezco / propongo / límite / para terapia)
- Ruta: separación / pausa estructurada / reintento condicionado
- Exportación de informe (imprimir / guardar como PDF)

## Cómo usar (local)
1. Descarga esta carpeta.
2. Abre `index.html` en tu navegador (Chrome/Edge/Firefox).

> Si quieres publicarla en GitHub Pages: sube estos archivos a un repo y activa Pages.

## Privacidad
- No hay servidor.
- Si activas “Guardar automáticamente”, se guarda en tu navegador (localStorage).
- Puedes borrar todo cuando quieras.

## Ideas de mejoras rápidas
- Exportar JSON (backup) e importar.
- Modo discreto (título/icono neutros).
- Más comparativas (texto y no solo escalas).
